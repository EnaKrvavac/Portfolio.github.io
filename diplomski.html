About Project

There are people who want to find investors, experienced business persons, a person with fresh ideas and so on... But most of them donâ€™t know where to look for those kinds of people. Moreover, creating a team with genuine people is really difficult and time-consuming. This application makes this process simpler. It allows users to find out interesting ideas. Users can search for expertise person and engage that person to work together on particular idea. Application will stimulate more people to collaborate together and make their dreams real. Nowadays, people are trying to provide IT solutions for every part of our lives. This application was started with the goal to be one of those solutions. It is an online application which gives opportunity to user to find investors, experienced business person or the person with fresh ideas. Users can search for or add new ideas. Users can register (create profile) in order to see all benefits of the application. Non-registered user cannot access application in order to keep user and other users project protected. The second part of a project allows users to see and update their profile information. All users can make profile. Registered users have ability to change password if it is forgotten. Using this application, users are able to meet new colleague as well as collaborate on different project ideas, without the need to leave their houses.
